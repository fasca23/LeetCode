**Подход: DFS со стеком**

1. **Идея:** Обходим дерево в глубину. Каждый элемент стека — пара (узел, глубина). На каждом шаге обновляем максимум. Детей добавляем с глубиной +1.
2. **Логика:** `stack = [(root, 1)]`. Пока стек не пуст: `node, depth = stack.pop()`. `max_depth = max(max_depth, depth)`. Если есть дети — `stack.append((child, depth + 1))`.
3. **Почему стек:** BFS с очередью тоже работает, но хранит целый уровень (O(w) памяти). DFS со стеком хранит только путь (O(h)). На широком дереве BFS может съесть гигабайты, DFS — нет.
4. **Время:** O(n) — каждый узел один раз.
5. **Память:** O(h) — стек в куче, не упадёт на вырожденном дереве.

**Ключевой момент:** Глубина хранится в стеке вместе с узлом. Не нужно вычислять её заново или хранить отдельно.
