**Подход: Динамическое программирование (числа Фибоначчи)**

1. **Идея:** На ступеньку `i` можно попасть либо с `i-1` (шаг +1), либо с `i-2` (шаг +2). Поэтому `ways(i) = ways(i-1) + ways(i-2)`. Это классические числа Фибоначчи.
2. **Логика:** База: `ways(1) = 1`, `ways(2) = 2`. Для i от 3 до n: `current = prev1 + prev2`, затем сдвигаем `prev2 = prev1`, `prev1 = current`.
3. **Почему Фибоначчи:** Каждый шаг зависит только от двух предыдущих. Вместо хранения всего массива dp[n] достаточно двух переменных.
4. **Особые случаи:** `n = 1` → 1. `n = 2` → 2. При n ≤ 2 сразу возвращаем n.
5. **Время:** O(n) — один проход от 3 до n.
6. **Память:** O(1) — только две переменные prev1 и prev2.

**Ключевой момент:** `prev2, prev1 = 1, 2` — начальные значения для ways(1) и ways(2). На каждой итерации окно сдвигается на один шаг вперёд.
