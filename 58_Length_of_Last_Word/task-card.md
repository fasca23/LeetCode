**Подход: Обход с конца**

1. **Идея:** Идём с конца строки. Сначала пропускаем все пробелы (если строка заканчивается пробелами). Затем считаем подряд идущие не-пробелы — это и есть последнее слово.
2. **Логика:** Два цикла `while`. Первый: `while s[i] == ' '` — декрементируем `i`. Второй: `while s[i] != ' '` — инкрементируем `length`, декрементируем `i`.
3. **Почему с конца:** Последнее слово может быть окружено пробелами. С начала пришлось бы разбивать всю строку. С конца — находим сразу.
4. **Особые случаи:** Пустая строка или только пробелы → `0`. Одна буква → `1`. Пробелы только в конце или только в начале — работает одинаково.
5. **Время:** O(n) — в худшем случае проходим всю строку.
6. **Память:** O(1) — две переменные.

**Ключевой момент:** Порядок циклов важен — сначала пропускаем пробелы, потом считаем буквы. Если поменять местами — сломается на строках с пробелами в конце.
