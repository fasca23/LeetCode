**Подход: Переворот половины числа**

1. **Идея:** Переворачиваем правую половину числа и сравниваем с левой.
2. **Логика:** На каждом шаге отщипляем последнюю цифру от `x` и добавляем к `rev`: `rev = rev * 10 + x % 10`, `x //= 10`. Останавливаемся, когда `x ≤ rev` (перевернули ровно половину).
3. **Особые случаи:** Отрицательные числа — всегда `false` (знак `-`). Числа, оканчивающиеся на 0 (кроме самого 0) — `false` (не может быть ведущего нуля).
4. **Сравнение:** Для чётной длины `x == rev` (например, 1221 → 12 == 12). Для нечётной `x == rev // 10` (например, 121 → 1 == 12 // 10), средняя цифра не влияет.
5. **Время:** O(log₁₀ n) — обрабатываем половину цифр числа.
6. **Память:** O(1) — только две переменные.

**Ключевой момент:** Останавливаем цикл когда `x ≤ rev` — это означает что мы перевернули ровно половину цифр, и готовы к сравнению.
