**Подход: BFS с очередью**

1. **Идея:** BFS обходит дерево по уровням. Первый встреченный лист даёт минимальную глубину. DFS пришлось бы проверять все пути.
2. **Логика:** `queue = deque([(root, 1)])`. Пока очередь: `node, depth = queue.popleft()`. Если лист (нет детей) → вернуть depth. Иначе добавить детей с depth+1.
3. **Почему BFS а не DFS:** BFS находит кратчайший путь первым. DFS может уйти в глубокую ветку, игнорируя мелкий лист в другой ветке.
4. **Время:** O(n) — в худшем случае обходим всё дерево.
5. **Память:** O(w) — ширина дерева (максимальное число узлов на одном уровне).

**Ключевой момент:** Лист = узел без обоих детей. Если есть только один ребёнок — это НЕ лист, путь идёт дальше.
