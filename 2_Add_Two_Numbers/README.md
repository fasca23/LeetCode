# Add Two Numbers

Сложение двух чисел в связных списках с цифрами в обратном порядке.  
**LeetCode #2**, сложность Medium.

## Задача

| Вход | Число | Выход | Сумма |
|------|-------|-------|-------|
| `[2,4,3] + [5,6,4]` | 342 + 465 | `[7,0,8]` | 807 |
| `[9,9,9] + [1]` | 999 + 1 | `[0,0,0,1]` | 1000 |
| `[0] + [0]` | 0 + 0 | `[0]` | 0 |

**Алгоритм**: один проход, поразрядное сложение с переносом. O(n).

python solution.py       # Демо
python test_solution.py  # Тесты
python visualize.py      # Визуализация

## Где применяется

-- Длинная арифметика — числа > 64 бит (криптография, финансы)
-- BigInt / BigDecimal — Python int, Java BigInteger
-- Процессоры — ripple-carry adder, поразрядное сложение
-- Потоки — данные приходят частями, длина неизвестна
-- Блокчейн — 256-битные хеши, base58-адреса
-- Финансы — точный расчёт без floating-point погрешностей

## Сложность

Время: O(max(n, m))
