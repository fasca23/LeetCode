**Подход: Поразрядное сложение с конца**

1. **Идея:** Идём справа налево по обеим строкам, складываем биты + carry. Текущий бит = `total % 2`, перенос = `total // 2`. Результат собираем в обратном порядке.
2. **Логика:** Указатели `i, j` с конца строк. На каждом шагу берём `bit_a` и `bit_b` (0 если строка кончилась). `total = bit_a + bit_b + carry` (может быть 0,1,2,3). Добавляем `str(total % 2)` в список, обновляем `carry = total // 2`.
3. **Почему с конца:** Сложение в столбик всегда начинается с младшего разряда (последняя цифра). Результат получается задом наперёд — в конце `reversed`.
4. **Особые случаи:** Разная длина — короткая строка добивается нулями. Финальный перенос — если после всех разрядов `carry == 1`, добавляем `"1"`. `"0" + "0"` = `"0"`.
5. **Время:** O(max(n, m)) — один проход.
6. **Память:** O(max(n, m)) — результирующая строка.

**Ключевой момент:** `while i >= 0 or j >= 0 or carry` — третье условие `carry` нужно, чтобы обработать финальный перенос (например, `"1" + "1"` → нужно добавить `"1"` после того как обе строки кончились).
