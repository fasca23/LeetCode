**Подход: DFS с перекрёстным сравнением**

1. **Идея:** Дерево симметрично если левое поддерево = зеркало правого. Запускаем DFS от `(root.left, root.right)`. На каждом шаге сравниваем значения. Детей добавляем перекрёстно: `(a.left, b.right)` и `(a.right, b.left)`.
2. **Логика:** `stack = [(root.left, root.right)]`. Пока стек не пуст: `a, b = stack.pop()`. Оба None → continue. Одно None или `a.val != b.val` → False. Иначе push перекрёстных пар.
3. **Почему перекрёстно:** Левое поддерево зеркально правому если крайний левый лист левого = крайнему правому листу правого, и так далее. Перекрёстный push это и обеспечивает.
4. **Время:** O(n) — каждый узел один раз.
5. **Память:** O(h) — стек в куче, не упадёт на вырожденном дереве.

**Ключевой момент:** `stack.append((a.left, b.right))` — левый ребёнок левого поддерева сравнивается с правым ребёнком правого. Именно это создаёт зеркальность.
