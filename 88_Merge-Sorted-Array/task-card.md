**Подход: Три указателя с конца**

1. **Идея:** Идём с конца, заполняя nums1 от наибольшего к наименьшему. Сравниваем `nums1[p1]` и `nums2[p2]`, больший ставим на `nums1[p]`.
2. **Логика:** `p = m+n-1`, `p1 = m-1`, `p2 = n-1`. Пока `p2 >= 0`: если `p1 >= 0` и `nums1[p1] > nums2[p2]` — берём из nums1, иначе из nums2. Уменьшаем соответствующие указатели.
3. **Почему с конца:** Если идти с начала, придётся сдвигать элементы nums1 чтобы освободить место. С конца место уже свободно (там нули-заполнители).
4. **Особые случаи:** `n = 0` → ничего не делаем. `m = 0` → просто копируем nums2. Элементы равны → берём из nums2 (любой порядок).
5. **Время:** O(m+n) — каждый элемент обрабатывается один раз.
6. **Память:** O(1) — только три указателя.

**Ключевой момент:** Когда `p2 < 0` — все элементы nums2 перенесены, останавливаемся. Оставшиеся элементы nums1 уже на своих местах.
