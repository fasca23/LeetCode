**Подход: Симуляция движения по строкам**

1. **Идея:** Идём по строке, раскладывая символы в список из `numRows` строк. Двигаемся вниз-вверх, меняя направление на границах.
2. **Логика:** Заводим список `rows` из пустых строк. `row` — текущая строка, `step` — направление (+1 вниз, -1 вверх). Каждый символ добавляем в `rows[row]`. На границах (`row == 0` или `row == numRows - 1`) меняем `step`. В конце склеиваем все строки.
3. **Почему симуляция:** Можно вычислить период и прыгать по индексам, но симуляция проще. Для `numRows == 1` — сразу возвращаем `s`.
4. **Особые случаи:** Одна строка или `numRows >= len(s)` — без изменений. Пустая строка — пустой результат.
5. **Время:** O(n) — один проход.
6. **Память:** O(n) — строки результата.

**Ключевой момент:** Направление меняется только на границах, в середине сохраняется.
