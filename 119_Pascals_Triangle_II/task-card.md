**Подход: Одна строка с обновлением справа налево**

1. **Идея:** Храним только текущую строку. Каждый новый ряд: добавляем 1 в конец, затем обновляем элементы от конца к началу: `row[j] = row[j] + row[j-1]`.
2. **Логика:** `row = [1]`. Для i от 1 до rowIndex: `row.append(1)`, для j от i-1 до 1: `row[j] += row[j-1]`.
3. **Почему справа налево:** Если идти слева направо, `row[j-1]` будет уже обновлённым значением текущего ряда, а нам нужно значение из предыдущего. Справа налево `row[j-1]` ещё не тронут.
4. **Время:** O(n²) — каждый элемент вычисляется один раз.
5. **Память:** O(n) — только одна строка.

**Ключевой момент:** Обновление справа налево — классический трюк для in-place обновления в задачах на треугольник Паскаля.
