**Подход: Одна строка с обновлением справа налево**

1. **Идея:** Храним только текущую строку. Каждый новый элемент = сумма двух верхних. Обновляем справа налево чтобы не затирать нужные значения.
2. **Логика:** `row = [1]`. Для i от 1 до rowIndex: добавляем 1 в конец, затем от j=i-1 до 1: `row[j] = row[j] + row[j-1]`.
3. **Время:** O(n²) — каждый элемент вычисляется один раз.
4. **Память:** O(n) — только одна строка.

**Ключевой момент:** Обновление справа налево позволяет использовать одну строку без доп. памяти.
