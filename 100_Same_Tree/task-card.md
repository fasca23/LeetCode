**Подход: DFS с явным стеком**

1. **Идея:** Вместо рекурсии — явный стек в куче. Храним пары узлов (p, q) для сравнения. pop → сравнили → push детей. Без риска RecursionError.
2. **Логика:** `stack = [(p, q)]`. Пока стек не пуст: `a, b = stack.pop()`. Оба None → continue. Одно None или `a.val != b.val` → False. Иначе `stack.append((a.left, b.left))`, `stack.append((a.right, b.right))`.
3. **Почему стек:** Рекурсия использует системный стек (~1000 вызовов в Python). Явный список в куче вмещает миллионы элементов. На вырожденном дереве-цепочке из 10 000 узлов рекурсия упадёт, стек — нет.
4. **Память:** O(h) — в стеке хранится путь от корня до текущего узла. Каждый элемент — кортеж из 2 ссылок (~16 байт).
5. **Время:** O(n) — каждый узел обрабатывается ровно один раз.

**Ключевой момент:** Порядок push: левый потом правый → pop достанет правый первым = DFS preorder. Порядок не важен для результата, важен для памяти (глубина вместо ширины).
